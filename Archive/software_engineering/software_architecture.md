### 모놀리식 아키텍처

- 전통적인 서버 사이드 시스템에서 사용하는 아키텍쳐
- 시스템의 모든 기능은 하나의 애플리케이션에서 작동한다.

1. 장점

- 개발이 더 빠르다 : 간단한 기능을 가진 애플리케이션으로 시작해 점점 기능을 추가할 수 있다.
- 성능이 더 좋다 : 마이크로 서비스와 달리 api 들을 통해 통신하는 오버헤드가 없어 빠르다.
